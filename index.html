Waarom volg ik deze cursus?

Oorspronkelijk ben ik een Directiesecretaresse, maar ik ben toevallig in deze rol gekomen en ik vind het erg leuk en interessant om met een website bezig te zijn. De afgelopen jaren heb ik de nodige cursussen gevolgd om mijn kennis up-te-daten, maar ik had nog niks gedaan met Webdesign. Ik merkte dat ik daar toch kennis te kort kwam. Daarom heb ik voor deze cursus gekozen.
